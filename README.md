# 0420_2

차량 디스플레이(1700x1184 비율) 기준의 **구간 태깅 내비게이션 프로토타입**입니다.

## 실행

정적 파일 기반이라 바로 열거나 간단 서버로 실행하면 됩니다.

```bash
python3 -m http.server 4173
```

브라우저에서 `http://localhost:4173` 접속.

## 프로토타입 플로우

1. `국민대학교 → 경복궁` 기본 경로 표시
2. 루트 1초 롱프레스 시 구간 개인화 모드 진입
3. 깃발 2개로 구간 선택 + 태깅 여부 확인
4. 확인 시 AI 에이전트 패널 슬라이드업
5. 음성/텍스트로 맥락 입력
6. 간단 LLM 해석(규칙 기반) 결과 표시
7. 지도에 태그 아이콘 고정
8. 회피 경로 재탐색 후 출발 프롬프트 표시

## 사용 API

- 지도 타일: OpenStreetMap
- 경로 API: OSRM demo (`https://router.project-osrm.org/route/v1/driving/...`)
